# clamp-number

[![experimental](http://badges.github.io/stability-badges/dist/experimental.svg)](http://github.com/badges/stability-badges)

Clamp value between range

## Usage

```js
var clamp = require('clamp-number')

clamp(2, 0, 3)  // == 2
clamp(10, 0, 3) // == 3
clamp(-5, 0, 3) // == 0
```
